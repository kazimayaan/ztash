STEP 1 : Logging in with Google

Why Google?


Secure Authentication:

  • ZTash Secrets Vault leverages Google's robust authentication system, known for its advanced security measures.
  • Google's authentication employs industry-leading practices to safeguard user accounts against unauthorized access.

STEP 2 : Managing Your Secrets Vault

Creating Ztash Secrets Vault:

  • After logging in, create a secure vault to organize and store your credentials.
  • Choose a unique vault name and, optionally, add a description for better organization.

STEP 3 : Adding Keys and Values

How to add keys?

  • Customize your vault by adding keys and corresponding values for different credentials.
  • This flexible structure allows you to manage a variety of sensitive information securely.

STEP 4: Get Token

How to Get Token?

  • Request a JSON Web Token (JWT) for secure access to your vault.
  • The token will be sent to your registered email, adding an extra layer of security.

STEP 5 : View Metrics

How to view Metrics?

  • Monitor your vault's usage and security metrics through the user-friendly dashboard.
  • Gain insights into key metrics, access history, and security-related data.
